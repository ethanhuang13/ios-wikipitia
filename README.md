# iOS 維基百坑

### 這本書是寫給誰的？

寫給具備一定 iOS 開發經驗，但是有許多觀念似懂非懂，或是想更精進的人。

如果沒有特別註明的話，書中的範例程式碼是用 Swift 3~4 來寫。

### 你在這本書會得到什麼？

關於 iOS 開發時可能會遇到的問題、工作流程，與心得分享。本書不強調系統化的學習，雖然篇章之間沒有太多關聯，但是你還是很有可能從這些零碎的知識獲得一些啟發。
